## v5.0.0

**BREAKING CHANGES**. The import style of this library has changed to support es-module style syntax.

#### Old code
```js
const kvstore = require('google-cloud-kvstore');
const store = kvstore(datastore);
```

#### New Code
```js
const {KVStore} = require('google-cloud-kvstore');
const store = new KVStore(datastore);
```
